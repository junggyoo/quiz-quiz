# Quiz quiz App
카테고리를 선택하고 간단한 퀴즈를 풀고 결과를 확인해보세요! 

 - 퀴즈를 풀면서 바로 정답과 오답을 확인 할 수 있고
 - 퀴즈 결과 화면에서 정답률, 소요시간, 정답 수, 오답 수를 확인할 수 있습니다. 
 - 오답 노트 기능도 제공합니다.

## 시작하기

```bash
npm install
npm run dev
```

로컬환경은 http://localhost:3000 에서 확인해 보실 수 있습니다.
